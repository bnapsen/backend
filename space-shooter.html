What changed

Better run flow

Shared realtime backend

The same websocket backend now powers chess, backgammon, and this co-op shooter, so hosting and invites feel consistent everywhere.

Playable survival loop

Waves escalate, pilots level up mid-run, boss dreadnoughts arrive on milestone waves, and downed teammates can come back if someone keeps the run alive.

Cleaner controls

Desktop uses mouse aim plus keys. Touch devices get a move stick, an aim pad that fires while held, and a separate boost trigger.

How to survive

Stay staggered

Two ships stacked together soak the same enemy pressure. Split the angle and cover each other.

Boost with intent

Boost helps you rescue a wingmate, dodge turret fire, or reach pickups before they expire.

Ride the pickups

Repair caches keep the squad alive. Overdrive powers up your spread shot and refills a chunk of shield.
