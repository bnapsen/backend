How to play locally

  1. Download the Windows zip package.
  2. Extract it somewhere on your PC.
  3. Open the Dragonfall-Jungle-Siege folder.
  4. Run INSTALL-DRAGONFALL.bat if you want the game copied into a local install folder with desktop and Start Menu shortcuts.
  5. You can also launch PLAY-DRAGONFALL.bat or MyProject9.exe directly from the extracted folder.
